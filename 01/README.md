# DOM 封装

## ★概念

- 封装：把复杂的东西打包成一个可以用简单命令操作的盒子
- 接口：被封装的东西暴露给外部的功能。
- 库：提供给其他人使用的工具代码叫做库（jQuery）
- API：库暴露出来的函数或属性
- 框架：当库变得很强大、并且需要学习才能看懂的叫做框架（Vue、React）

jQuery 的基本设计思想和主要用法，就是"**选择某个网页元素，然后对其进行某种操作**"。这是它区别于其他 Javascript 库的根本特点

> 指定某个节点，对其搞事情！
