## ★介绍

- ES 6 新特性列表：<https://frankfang.github.io/es-6-tutorials/>

如果你想自学，可以直接看上面两个免费链接学习。

不过，如果你购买这本课程，可以看看一个使用 JS 五年以上的人是怎么理解这些特性的（包含大部分书籍上都看不到的实战经验）。

讲解过程中会横向对比 Python、Ruby、CoffeeScript 等其他语法的语法，以便加深理解。
